# GitHub Profile Finder

A simple and elegant web application that allows users to **search for any GitHub profile** by entering a valid username. It fetches real-time profile data using the GitHub REST API and displays useful information like the userâ€™s name, avatar, bio, number of public repositories, followers, and following count.

-  Show followers and following count
